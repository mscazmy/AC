# بارد — موقع خدمة تصليح مكيفات الهواء في العراق

موقع عربي احترافي لخدمة تصليح وصيانة مكيفات الهواء، مع لوحة تحكم للمسؤول.

## 🚀 نشر الموقع على GitHub Pages

1. أنشئ مستودعاً جديداً على GitHub (مثال: `barid-ac`)
2. ارفع جميع الملفات إلى المستودع
3. اذهب إلى: **Settings → Pages → Source → GitHub Actions**
4. سيتم النشر تلقائياً عند كل تحديث

الموقع سيكون متاحاً على: `https://yourusername.github.io/barid-ac/`

---

## 🔑 لوحة التحكم

**الرابط:** `/admin/`  
**المستخدم:** `admin`  
**كلمة المرور الافتراضية:** `admin123` ← **غيّرها فوراً من قسم الإعدادات**

### الميزات:
- ✅ نشر مقالات ونصائح تظهر في الموقع
- ✅ إضافة صور لمعرض الأعمال (رفع مباشر أو رابط)
- ✅ تغيير صورة البانر الرئيسي
- ✅ تغيير كلمة مرور المسؤول

---

## ☁️ ربط Supabase (مستقبلاً)

1. أنشئ مشروعاً على [supabase.com](https://supabase.com)
2. أنشئ جدولين: `posts` و `gallery`
3. أضف مفاتيح API في `/admin/dashboard.html` في قسم Supabase
4. استبدل دوال `localStorage` بطلبات Supabase API

---

## 📂 هيكل الملفات

```
/
├── index.html          ← الصفحة الرئيسية
├── css/style.css       ← التنسيقات
├── js/main.js          ← السكريبت الرئيسي
├── images/hero.png     ← صورة البانر الافتراضية
├── admin/
│   ├── index.html      ← صفحة تسجيل الدخول
│   └── dashboard.html  ← لوحة التحكم
└── .github/workflows/deploy.yml ← نشر تلقائي
```

## تخصيص الموقع

- **رقم الهاتف:** ابحث عن `07X XXXX XXXX` واستبدله برقمك في `index.html`
- **واتساب:** عدّل رابط `wa.me/` في `index.html`
- **الشعار:** ابحث عن `بـــارد` لتغيير اسم الشركة
- **المحافظات:** عدّل قائمة `<select>` في نموذج التواصل
